# Air Quality Analysis Project

## Overview
This project explores air quality data across Indian cities using exploratory data analysis and basic machine learning techniques. The focus is on pollutant distributions, AQI trends, and predicting AQI from key pollutants.

## Contents
- Indian_AQI_Data_Analysis.ipynb: Data cleaning, visualisations, EDA, and AQI prediction model.
- Streamlit dashboard prototype for interactive exploration.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
